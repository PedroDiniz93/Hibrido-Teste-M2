## Sobre
Modulo de cores, altera a cor do botão por comando

---

## Configuração (Como instalar o módulo)

1. Ativar o módulo.
2. Rode o comando (CLI) /bin/magento color-change --color '0317fc' --store 1 para que altere a cor do botão
